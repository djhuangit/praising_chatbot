## Getting Started

### Prerequisites
- Python 3.8 or higher
- [uv](https://docs.astral.sh/uv/) (recommended) or pip
- OpenAI API key
- PostgreSQL (for production) or SQLite (for development)

### Installation

#### Option 1: Using uv (Recommended - Much Faster! âš¡)

1. Install uv if you haven't already:
```bash
# On macOS and Linux
curl -LsSf https://astral.sh/uv/install.sh | sh

# On Windows
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"

# Or with pip
pip install uv
```

2. Clone the repository:
```bash
git clone [repository-url]
cd praising_chatbot
```

3. Install dependencies with uv:
```bash
uv sync
```

4. Create a `.env` file:
```bash
cp .env.example .env
```

Edit the `.env` file with your credentials:
```
OPENAI_API_KEY=your-openai-api-key-here
DATABASE_URL=sqlite+aiosqlite:///./chat.db  # Or your PostgreSQL URL
```

5. Run the application:
```bash
uv run python app.py
```

#### Option 2: Using pip (Traditional)

1. Clone the repository:
```bash
git clone [repository-url]
cd praising_chatbot
```

2. Create a virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: .\venv\Scripts\activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Create a `.env` file:
```bash
cp .env.example .env
```

Edit the `.env` file with your credentials:
```
OPENAI_API_KEY=your-openai-api-key-here
DATABASE_URL=sqlite+aiosqlite:///./chat.db  # Or your PostgreSQL URL
```

5. Run the application:
```bash
python app.py
```

### Running the Application

**With uv (recommended):**
```bash
uv run python app.py
```

**With pip:**
```bash
python app.py  # Make sure your venv is activated
```

The application will be available at:
- **Local**: http://localhost:7860
- **Network**: http://0.0.0.0:7860

## Usage

1. Open the application in your browser
2. Type your message in the text box
3. Click "Send" or press Enter
4. Receive supportive and encouraging responses!
5. Track your API usage in real-time at the top of the page

## Configuration

### Database Options

**SQLite** (Default - Easy for development):
```
DATABASE_URL=sqlite+aiosqlite:///./chat.db
```

**PostgreSQL** (Production):
```
DATABASE_URL=postgresql+asyncpg://user:password@host:port/database
```

### OpenAI Model

Currently using `gpt-4o-mini` at $0.15 per 1M tokens. To change the model, edit `app.py` line 76.
